# Simple Ad Blocker

Chrome(및 크로미움 계열 브라우저)용 오픈소스 광고 차단 확장 프로그램입니다.
Manifest V3의 `declarativeNetRequest` API를 사용해 광고/추적 서버 요청을
브라우저 단계에서 차단하고, 유튜브의 영상·이미지 광고를 자동으로 건너뜁니다.

크롬 웹스토어의 정책상 이런 유형의 확장은 등록이 제한되므로,
이 프로젝트는 **웹스토어를 거치지 않고 직접(sideload) 설치**하는 방식으로 배포됩니다.
이는 오픈소스 광고 차단기에서 일반적인 배포 방식입니다.

## 구성

| 파일 | 역할 |
|------|------|
| `manifest.json` | 확장 정의 (권한·규칙·스크립트 등록) |
| `rules.json` | 차단할 광고/추적 도메인 규칙 (자동 생성) |
| `hide_ads.css` | 차단 후 남는 광고 빈 자리 숨김 |
| `youtube_ads.css` | 유튜브 피드/사이드바 광고 숨김 |
| `youtube_skip.js` | 유튜브 영상·이미지 광고 자동 스킵 |

## 설치 방법 (사용자용)

1. 이 폴더(`Simple-Ad-Blocker`)를 압축 해제해 원하는 위치에 둡니다.
   (설치 후에도 이 폴더는 삭제하면 안 됩니다.)
2. Chrome 주소창에 `chrome://extensions` 입력 후 이동
3. 오른쪽 위 **개발자 모드** 스위치 켜기
4. **압축해제된 확장 프로그램을 로드합니다** 클릭
5. 이 폴더를 선택

설치 후 브라우저를 재시작하면 적용됩니다.

## 차단 목록 갱신

광고 도메인 목록은 시간이 지나면 낡습니다. 저장소 루트의 `make_rules.py`를
실행하면 최신 목록으로 `rules.json`을 다시 생성합니다. 그 후
`chrome://extensions`에서 이 확장을 새로고침하세요.

## 한계 (솔직히)

- 유튜브가 광고를 영상에 서버에서 미리 합쳐 보내는 방식(SSAI)은 차단 불가
- 완전한 0초 스킵은 불가능 (광고 스트림이 뜨는 시간 자체는 존재)
- 유튜브가 "광고 차단기 사용 중" 경고를 띄울 수 있음
